# zeromq

ZeroMQ core engine in C++, implements ZMTP/3.1

## Maintainers

* The Habitat Maintainers: <humans@habitat.sh>

## Type of Package

Binary package

## Usage

Include `core/zeromq` in your `plan.sh` so that your apps can compile against the `libzmq`.
